# FR MOLETOM - Premium E-Commerce Frontend

A high-performance, modern, and professional e-commerce ecosystem custom-built for **FR MOLETOM**, a brand specialized in premium children's menswear (ages 2 to 16). This repository showcases a secure, production-ready frontend architecture built with HTML5, CSS3, Modern JavaScript, and Bootstrap 5.

---

## 🔒 Source Code Protection & Compliance Notice

This public repository contains a **sanitized showcase version** of the project. To comply with non-disclosure agreements (NDA), protect proprietary business logic, and ensure the company's cybersecurity integrity, several enterprise-level modules have been intentionally omitted or modified from this public code:

* **Authentication & Identity:** Enterprise features such as secure MFA (Multi-Factor Authentication), full JWT login validation, and password recovery workflows have been decoupled and stripped.
* **Database & Back-end Layer:** All direct database hooks, internal API endpoints, and server-side configurations are hidden to protect corporate cloud infrastructure.
* **Admin Dashboard:** The administrative panel included here has been simplified into an exhibition-only layout, allowing recruiters and developers to safely inspect its visual architecture and UX workflows without exposing real operational data.

The live, fully integrated system deployed by the company runs on a private network featuring isolated database management, encrypted webhooks for payment processing (Mercado Pago, Pix), and robust security protocols.

---

## 🚀 Key Features (Public Showcase)

* **Premium UI/UX:** Tailored strictly to the brand’s visual identity using Deep Blue and High-Contrast Orange.
* **Performance Optimized:** Semantic HTML5 and modular Vanilla JS designed for fast rendering and high conversion rates.
* **Fully Responsive:** Micro-interactions, automatic carousels, and layouts optimized for Mobile, Tablet, and Desktop.
* **Future-Ready Architecture:** Clean DOM mapping prepared for seamless headless API integrations (Shopify, WooCommerce, or custom REST APIs).
